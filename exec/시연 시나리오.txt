[B309팀 시연 시나리오]

1. 메인페이지 
- 로그인 전 화면 일반적
- 로그인 소개, 로그인 후 조회수 top10 기업 보임
- 로그인 후 없던 navbar 애들 말하고 기업추천 클릭

2. 기업 리스트 페이지
- 기업리스트 확인

3. 기업추천
-  기업추천은 5가지 지표를 활용해 16가지 특징을 가진 가상기업목록을 만들어 토너먼트를 진행
- 사용자가 기업을 선택할시 재무지표에서 어느 부분을 중요시 여기는지 확인하고 해당 특징과 
가장 유사한 지표특징을 가진 기업을 추천해줌

4. 추천 기업 중 1 클릭 (청년친화기업 상세 페이지)
- 기업 정보 설명 : 시각화 보면서 토너먼트 우승 기업과 비슷한 구조의 그래프 모양을 확인가능
- 기업 상세 페이지를 오면 청년친화강소기업인 경우 추가 정보를 확인할 수 있으며 
- 재무정보와 아래에 기업의 특허데이터를 확인 할 수 있어 해당 기업이 어느 분야에 장점이 있는지
개인이 판단할 수 있는 요소를 만들어 놓았습니다.
